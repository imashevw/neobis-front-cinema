# MovieApp

MovieApp - это веб-приложение для поиска, просмотра информации о фильмах, а также отслеживания последних премьер и релизов.

## Установка

1. Склонируйте репозиторий: `git clone https://github.com/imashevw/neobis-front-cinema.git`
2. Откройте файл index.html в вашем браузере.

## Использование

1. Введите поисковый запрос в поле "Search" для поиска фильмов.
2. Используйте навигационные кнопки для просмотра различных категорий фильмов.

## Структура проекта

- **index.html**: Главный файл HTML с разметкой и подключением стилей и скриптов.
- **style.css**: Файл стилей для оформления внешнего вида приложения.
- **script.js**: JavaScript-скрипт для взаимодействия с API и обработки событий.

## Навигационные кнопки

- **Премьеры этого месяца**
- **Топ ожидаемых фильмов**
- **Релизы этого месяца**
- **Топ лучших фильмов**
- **Твои избранные фильмы**
